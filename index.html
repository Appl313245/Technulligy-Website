Project Title

This is a project for NWAPW modifies vanilla minecraft so that you can enhance your tools.

We use Java as our scripting language. Technulligy is a Minecraft mod that allows Minecraft players to enhance their tools using cores. If you are a Minecraft player who wishes you can make any tool become stronger than technulligy is the mod for you. In the early stages of Minecraft, you are stuck with stone tools and maybe some iron. But everything you do is so slow and it seems as if you are making little progress. Technulligy enables a user to increase their tool’s statistics with cores. Which will allow players to get through the boring early stages faster. During the late game stages of Minecraft, once you get the strongest resource available, you can’t progress anymore which is really frustrating. With Technulligy, you can get even stronger in the late game with unstable cores to make yourself stronger. Be careful though, with unstable cores, there is a chance your core can turn into a useless null core. In addition, you can create many custom tools that can help you along the journey.
